1) В меню программы пользователь выбирает вариант для генерации пароля 
    1.1) Пользователю надо будет ввести символы с клавиатуры и из них он получит пароль
    1.2) Пользователю надо записать символы в файл и указать адрес на этот файл(адрес можно вводить как абсолютный, так и относительный )
    1.3) Пользователь получит пароль из символов по умолчанию (при выборе 3 пункта эти символы будут выведены на экран)

2) Перед функциями я использовал декоратор для заера работы времни функций для генерации пароля(но не всей программы)
